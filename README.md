# ANN_Project_GroupB_AI_6A1

Real-Time Market Movement Prediction System using Sequential Deep Learning models (RNN, LSTM, GRU, and BiLSTM). The project combines stock market technical indicators with financial news sentiment analysis to predict market direction (Up/Down).

## Features

* Real-time stock data using Yahoo Finance
* Financial news sentiment analysis using VADER
* Technical indicator generation
* Deep Learning models: RNN, LSTM, GRU, BiLSTM
* Flask API deployment
* Interactive Gradio web interface
